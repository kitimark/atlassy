# Atlassy Ideas

This folder stores incubation notes, promoted-source idea docs, and supporting evidence notes.

## Purpose

- Capture useful concepts without forcing them into the active roadmap.
- Keep idea documents small and focused for easier AI-assisted discussion.
- Define clear signals for when an idea should be promoted to `roadmap/` and preserve historical context after promotion.

## Idea Document Template

Use this structure for each idea:

- `Status`
- `Plain Problem Points`
- `Proposed Direction`
- `Why Not Now` (or `Historical Why Not Now (pre-promotion)`)
- `Risks`
- `Signals To Revisit`
- `Promotion Path` (or `Promotion Outcome`)
