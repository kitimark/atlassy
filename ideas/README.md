# Atlassy Ideas

This folder stores project ideas that are not scheduled for implementation yet.

## Purpose

- Capture useful concepts without forcing them into the active roadmap.
- Keep idea documents small and focused for easier AI-assisted discussion.
- Define clear signals for when an idea should be promoted to `roadmap/`.

## Idea Document Template

Use this structure for each idea:

- `Status`
- `Plain Problem Points`
- `Proposed Direction`
- `Why Not Now`
- `Risks`
- `Signals To Revisit`
- `Promotion Path`
